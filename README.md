# 👗 URBAN – Modern Fashion Landing Page

A premium, fully responsive e-commerce landing page for a contemporary fashion brand. Built with **Vanilla HTML, CSS & JavaScript**.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎠 **Product Carousel** | Horizontal slider with 6 products & touch support |
| 📜 **Marquee Text** | Endless moving text banner for urban vibe |
| 🌀 **3D Diorama** | Floating 3D hero element with parallax effect |
| 🌟 **Gradient Text** | Animated gradient on headlines |
| ⭐ **Reviews** | 5-star customer testimonials |
| 📬 **Newsletter** | Email subscription section |
| ⬆️ **Back-to-Top** | Smooth scroll-to-top button |
| 📱 **Fully Responsive** | Perfect on all devices |
| ✨ **Scroll-Reveal** | Cards appear on scroll |
